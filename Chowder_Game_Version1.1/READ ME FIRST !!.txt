
  - How To Install / Informatii:
  - Ruleaza "Chowder_Launcher.exe" fara a fi nevoie de installer (RECOMANDAT) SAU
  - Ruleaza "setup.exe"

  	Observatie: -dupa instalare jocul va porni automat
		    -jocul are volum
	
  - Launcher-ul este "Chowder_Launcher.application"
  - Rezolutia optima pentru joc este 1440 x 900 deoarece aceasta este rezolutia la care a fost realizat jocul

	Observatie: -orice rezolutie mai mare de 1440 x 900 este buna
		    -orice rezolutie sub 1400 x 900 NU este buna

  - Pentru dezinstalare mergeti la "Control Panel" -> "Uninstall a program" -> "Chowder_Launcher" ("Uninstall")
  - Enjoy !!!
